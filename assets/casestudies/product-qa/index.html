PRODUCT QA

AI Powered PDF Query Automation

I built an AI powered automation agent that helps enterprises manage unstructured PDF data by enabling real time extraction, organization, and intelligent querying directly through Telegram. Using Make.com as the automation backbone, I integrated Google Drive for document handling, DumplingAI for PDF to text conversion, and Google Gemini AI for advanced analysis.

To deliver precise results, I implemented Retrieval Augmented Generation (RAG), which provides contextual and accurate responses to user queries. I also automated the process of formatting and storing extracted text in Google Sheets, ensuring easy reference and transparency. This solution reduced response times and manual lookups by 70%, while offering a scalable, low maintenance system that accelerates decision making, improves efficiency, and enables businesses to handle large volumes of documents seamlessly.
